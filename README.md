## 📁 Files

- `index.html` - Main HTML file
- `style.css` - All styling
- `script.js` - JavaScript for interactions
- `README.md` - This file

## 🔧 Setup

1. **Download** all three files (index.html, style.css, script.js)
2. **Place them in the same folder**
3. **Open index.html** in your browser
4. **That's it!** Your portfolio is ready!

## 🌐 Deployment

### Netlify Drop (Easiest - 30 seconds)
1. Go to https://app.netlify.com/drop
2. Drag all 3 files into Netlify
3. Get your live link instantly!

### GitHub Pages
1. Create a new repository
2. Upload all files
3. Enable GitHub Pages in settings
4. Get your live URL: `https://yourusername.github.io/portfolio`

### Vercel
1. Go to vercel.com
2. Import your files
3. Get auto-deployed live link

## 🎨 Customization

### Change Colors
Edit `:root` variables in `style.css`:
```css
--accent: #4f8cff;        /* Blue highlight color */
--bg: #0a0e14;            /* Background */
--text: #e6e8eb;          /* Text color */
```

### Change Text
Edit any text directly in `index.html`

### Add/Remove Sections
Copy-paste section blocks in `index.html` and create matching CSS in `style.css`

### Change Profile Photo
Replace the base64 image in index.html with your own photo converted to base64
